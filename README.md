# TutorMatch

TutorMatch is a React Native / Expo tutoring marketplace app where students can browse tutors, book sessions, and chat in realtime.

## Features

- Supabase authentication
- User profiles
- Tutor search and filters
- Booking system
- Booking cancellation
- Realtime chat
- Message previews
- Pull-to-refresh
- Reusable loading/error/empty states

## Tech Stack

- Expo
- React Native
- TypeScript
- Expo Router
- Supabase Auth
- Supabase Database
- Supabase Realtime
