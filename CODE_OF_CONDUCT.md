# Code of Conduct

## Purpose

This repository is maintained as a professional portfolio and demonstration project.

All participants and viewers are expected to engage respectfully and professionally.

## Expected Behavior

Please:
- be respectful
- provide constructive feedback
- avoid harassment or abusive language
- respect licensing restrictions

## Prohibited Behavior

The following is prohibited:
- harassment or discrimination
- unauthorized reproduction of project code
- redistribution of proprietary material
- misuse of repository content

## Enforcement

The repository owner reserves the right to:
- remove comments/issues
- block users
- report misuse or unauthorized redistribution

## Contact

For concerns or questions, contact the repository owner.
